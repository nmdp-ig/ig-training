Official URL: http://example.org/ig/exampleig/ValueSet/gvhd-all-valueset Version: 0.2.0
Active as of 2023-05-02 Computable Name: GVHDTypeValueSet

Permissible values for types of Graft vs Host Disease, from SNOMED CT and ICD-10

References

  • GVHDDiagnosis

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Import all the codes that are contained in GVHD SNOMED CT Codes
  • Import all the codes that are contained in GVHD ICD-10 Codes
